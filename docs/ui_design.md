# FilmCam Assist — UI 设计 v2

## 设计原则

1. **竖屏优先** — 摄影师单手操作手机，另一手持相机
2. **统一交互** — 所有参数调节采用同一种交互：触按滑动调值，松手刻度消失
3. **摄影工作流** — 按 Film → 曝光 → 色彩 → 距离 组织，而非 App 功能
4. **大 UI 元素** — 户外强光下也能快速读数，手指粗也能精准操作

---

## 核心测光逻辑

### 输入 vs 输出

| 模式 | 用户输入（可调） | 只读输出（自动算） | 曝光补偿作用 |
|------|----------------|-------------------|-------------|
| **A** | ISO + 光圈 + EV | 快门 | EV → 偏移快门计算 |
| **S** | ISO + 快门 + EV | 光圈 | EV → 偏移光圈计算 |
| **M** | ISO + 光圈 + 快门 | （无） | EV 不参与；显示测光读数与实际设置的偏差 |

### 说明

- **底部导航已移除** — 不再有 Meter/Range/Color/Frame 四个 tab
- 所有功能整合为四个纵向小节（Film / 曝光 / 色彩 / 距离）
- 预览区下方是一个可纵向滚动的设置面板
- 每个小节的标题可点击折叠/展开
- 每个设置项（除距离为只读）采用统一交互

## 统一交互方式：滑尺（ValueWheel）

所有可调参数共用同一套交互：

```
┌────────────────────────────┐
│  状态：缩起                │
│  ISO  400          ▸      │  ← 显示当前值 + 右箭头
└────────────────────────────┘
         ↓ 点击/触按
┌────────────────────────────┐
│  状态：展开                │
│  ISO  400          ▸      │  ← 值不变
│  ┌──────────────────────┐ │
│  │ 200  400*  800  1600 │ │  ← 水平刻度条
│  │                      │ │     当前值居中高亮
│  └──────────────────────┘ │     左右滑动选值
│                           │     松手后刻度消失
└────────────────────────────┘
```

详细规则：

| 规则 | 说明 |
|------|------|
| 触发 | 点击/触摸参数行 |
| 刻度出现 | 在参数行下方展开一个水平刻度条，**跨整行宽度**（覆盖两列）|
| 滑动 | 左右横向滑动 |
| 步进 | 整档（可轻滑 1/3 档）|
| 反馈 | 滑动时值实时更新，当前值高亮 |
| 消失 | 松手后刻度条缩回（或点击其他区域） |
| 重入 | 再次点击参数行重新展开 |
| 多行 | 同一时间只有一个参数的刻度条展开，展开其他参数会自动收起当前 |

刻度条样式：
- 背景半透明黑色（如 `Colors.black.withAlpha(200)`）
- 数值横向排列，当前值 cyanAccent 高亮
- 尺寸随屏幕宽度等比缩放（参考 ScreenScale）

---

## 小节布局规则

- 每个小节的参数按 **每行两项、均分宽度、左对齐** 排列
- 每项宽度 = (可用宽度 − 间距) / 2
- 参数行在折叠状态下显示当前值 + `▸` 箭头
- 点击任意行展开该参数的 ValueWheel 刻度条（独占整行宽度，跨两列）

## 各小节内容

### Film（胶片）— 1 行

| 左列 | 右列 |
|------|------|
| **画幅** — 选择器：135(35mm) / 120(6×4.5) / 120(6×6) / 120(6×7) / 120(6×9) / 4×5" LF | **ISO** — 滑尺：25/50/100/200/400/800/1600/3200/6400 |

### 曝光（Exposure）— 3 行

| 左列 | 右列 |
|------|------|
| **测光模式** — 循环点按：⬤ Spot / ●○ CW / ◐ Avg | **档位** — 循环点按：A / S / M |
| **光圈** — 滑尺：1.0–22 标准档 | **快门** — 滑尺：30"–1/8000 |
| **EV** — 滑尺：-3.0 ~ +3.0 (1/3 step) | （空）|

A/S/M 各模式下各参数的输入/输出状态：

| 参数 | A | S | M |
|------|---|---|---|
| 光圈 | **输入** | 只读 | **输入** |
| 快门 | 只读 | **输入** | **输入** |
| EV | **输入** | **输入** | 只读偏差 |

### 色彩（Color）— 1 行

| 左列 | 右列 |
|------|------|
| **色温 (Temp)** — 滑尺：2000K–10000K | **Tint** — 滑尺：-150 ~ +150 |

### 距离（Distance）— 1.5 行

| 左列 | 右列 |
|------|------|
| **当前距离** — 只读显示（来自 Camera2 AF / LiDAR） | **近界** — 只读显示（根据光圈/焦距/画幅计算） |
| **远界** — 只读显示 | （空）|

## 相机预览 (Camera Preview)

- 固定 4:3 裁切比例（`AspectRatio(4/3)`）
- 竖屏时宽度撑满，高度按 4:3 计算
- 叠加显示：
  - 测光点（黄色圆圈，点测光时可见）
  - 框线（根据画幅/焦距）
  - 网格线（三分法）
- 点按预览可移动测光点

---

## 视觉风格

- **深色主题** — 黑色背景（`Colors.black`）
- **配色**：
  - 白色 — 可调输入值 / 章节标题
  - 灰色（`white38`/`white54`）— 标签 / 说明文字
  - 青色（`cyanAccent`）— 高亮选中值、标尺指示
  - 橙色 — EC 非零 / 曝光警告
  - 绿色 — 曝光正确 / 数据就绪
- **字体** — 等宽数字（`FontFeature.tabularFigures()`）
- **最小触控目标** — 48×48 pt
- **响应式** — 所有尺寸通过 `ScreenScale` 等比缩放（参考宽度 375pt）
