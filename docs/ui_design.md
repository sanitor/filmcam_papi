# FilmCam Assist — UI 设计

## 设计原则

1. **竖屏优先** — 摄影师单手操作手机，另一手持相机
2. **大 UI 元素** — 户外强光下也能快速读数，手指粗也能精准操作
3. **胶片摄影师的思维模型** — 不按"App 功能"分组，按"摄影工作流"组织
4. **物理感** — 光圈/快门的调节像转动镜头环或相机拨盘

---

## 核心测光逻辑

### 输入 vs 输出

| 模式 | 用户输入（可调） | 只读输出（自动算） | 曝光补偿作用 |
|------|----------------|-------------------|-------------|
| **A** | ISO + 光圈 + EV | 快门 | EV → 偏移快门计算 |
| **S** | ISO + 快门 + EV | 光圈 | EV → 偏移光圈计算 |
| **M** | ISO + 光圈 + 快门 | （无） | EV 不参与；显示测光读数与实际设置的偏差 |

### 影响计算的参数

- **ISO** — 每卷胶卷固定，最优先设置
- **画幅（Format）** — 影响景深计算（弥散圆 CoC 取决于画幅），以及取景框线
- **焦距** — 影响取景框线和视差

## 各区域详细设计

### 1. 相机预览 (Camera Preview)
- 顶部 ~30% 屏幕，4:3 裁切适配竖屏
- 叠加显示：测光点（黄色圆圈）、斑马纹（过曝提示）、框线（取景辅助）
- 点按可切换测光点位置（点测光模式）
- 画幅和焦距影响框线显示

### 2. 曝光补偿标尺 / 偏差指示 (EC Scale)

**A / S 模式** — 曝光补偿输入：
- 标尺：`-3  -2  -1  0  +1  +2  +3`
- **左右滑动**调节补偿值，1/3 EV 步进
- 当前值大字显示 `0.0` / `-0.7` / `+1.3`
- 补偿非零时值变橙色
- 效果：EC → 偏移自动计算的那个参数

**M 模式** — 测光偏差指示（只读）：
- 同一标尺，但不可滑动
- 指针位置 = 测光 EV - 用户设置 EV
- 归零 = 曝光正确

### 3. 光圈 / 快门控制

两个竖向大数值，根据模式决定输入/输出：

| 模式 | 左（光圈） | 右（快门） | 交互 |
|------|-----------|-----------|------|
| **A** | 可调（亮色大字） | 只读（灰色小字） | 上下滑动调光圈 |
| **S** | 只读（灰色小字） | 可调（亮色大字） | 上下滑动调快门 |
| **M** | 可调（亮色大字） | 可调（亮色大字） | 上下滑动分别调 |

交互：
- **上下滑动**数值区域改变数值
- 整档步进，轻滑 1/3 档
- **点按**弹出滚轮精确选择

档位标准：
- 光圈：1.0, 1.1, 1.2, 1.4, 1.6, 1.8, 2.0, 2.2, 2.5, 2.8, 3.2, 3.5, 4.0, 4.5, 5.0, 5.6, 6.3, 7.1, 8.0, 9.0, 10, 11, 13, 14, 16, 18, 20, 22
- 快门：30" 15" 8" 4" 2" 1" 1/2 1/4 1/8 1/15 1/30 1/60 1/125 1/250 1/500 1/1000 1/2000 1/4000 1/8000

### 4. 参数栏 (Meter Bar)

一行显示四个参数，**点击切换**：

| 参数 | 显示 | 点击交互 |
|------|------|----------|
| **ISO** | `ISO 400` | 弹出感光度滚轮：25/50/100/200/400/800/1600/3200/6400 |
| **测光模式** | `⬤`(点测) / `●○`(中央) / `◐`(平均) | 循环切换 |
| **优先模式** | `A` / `S` / `M` | 循环切换 A→S→M→A |
| **焦距 / 画幅** | `50mm` / `135` | 点击展开设置（也可在设置面板改） |

### 5. 底部导航 (Mode Tabs)

四个大按钮带 icon（48×48 min）：

| 按钮 | 功能 |
|------|------|
| ☀ Meter | 测光模式（主界面，默认） |
| ⊥ Range | 测距 + 景深标尺 |
| ◐ Color | 色温 (CCT + Duv) |
| ⊞ Frame | 取景框线 + 视差校正 |

## 用户初始设置流程

第一次打开 App：

1. 相机权限授权
2. 设置页引导：
   - 你现在用什么胶卷？→ ISO 选择
   - 什么画幅？→ 135 / 120 (645/6×6/6×7/6×9) / 4×5
   - 镜头焦距？→ 50mm（默认）
3. 进入主 Meter 模式，可直接开始测光

---

## 设置面板

从主屏下滑或点齿轮图标打开：

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 焦距 | 50mm | 取景框线 + 景深计算 |
| 画幅 | 135 (35mm) | 135 / 120(645/6×6/6×7/6×9) / 4×5 |
| 校准 - EV 偏移 | 0.0 | 测光整体校准 |
| 校准 - CCT 偏移 | 0K | 色温校准 |
| 校准 - 距离偏移 | 0.0m | 测距校准 |

## 视觉风格

- **深色主题** — 黑色背景，减少户外反光干扰
- **配色**：
  - 白色亮色 — 可调输入值
  - 灰色 — 只读输出值 / 标签
  - 橙色 — EC 非零 / 警告
  - 绿色 — 读数正常 / 数据就绪
- **字体** — 等宽数字（tabular figures）
- **最小触控目标** — 48×48 pt
